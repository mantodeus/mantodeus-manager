## Getting Started

### Logging In

1. Navigate to your Mantodeus Manager URL
2. Click the **Login** button
3. Authenticate using your credentials
4. You'll be redirected to the Jobs dashboard

### Dashboard Overview

The dashboard provides quick access to:
- **Jobs**: View and manage all construction projects
- **Reports**: Generate and download project reports
- **User Profile**: Access your account settings

## Managing Jobs

### Creating a New Job

1. Click the **New Job** button on the Jobs page
2. Fill in the job details:
   - **Job Title** (required): Name of the construction project
   - **Description**: Brief overview of the project
   - **Location**: Physical address or site location
   - **Status**: Current project status (Planning, Active, On Hold, Completed, Cancelled)
   - **Start Date**: Project start date
   - **End Date**: Expected completion date
3. Click **Create Job**

### Viewing Job Details

1. Click on any job card from the Jobs list
2. The job detail page displays:
   - Job information (title, description, location, dates)
   - Status badge
   - Three main tabs: Tasks, Images, Reports

### Editing a Job

Job editing functionality can be added by administrators through the job detail page. Contact your system administrator for job modifications.

### Job Status Types

- **Planning**: Project is in planning phase
- **Active**: Construction is actively underway
- **On Hold**: Project temporarily paused
- **Completed**: Project finished
- **Cancelled**: Project cancelled

## Working with Tasks

### Creating a Task

1. Open a job detail page
2. Navigate to the **Tasks** tab
3. Click **New Task**
4. Fill in task details:
   - **Task Title** (required): Clear description of the task
   - **Description**: Detailed requirements and notes
   - **Status**: Current task status (To Do, In Progress, Review, Completed)
   - **Priority**: Task priority level (Low, Medium, High, Urgent)
   - **Assign To**: Select team member to assign the task
   - **Due Date**: Task deadline
5. Click **Create Task**

### Task Status Workflow

Tasks follow a standard workflow:

1. **To Do** → Click **Start** → **In Progress**
2. **In Progress** → Click **Review** → **Review**
3. **Review** → Click **Complete** → **Completed**

### Task Priority Levels

- **Low**: Non-urgent tasks that can be scheduled flexibly
- **Medium**: Standard priority for regular work items
- **High**: Important tasks requiring prompt attention
- **Urgent**: Critical tasks needing immediate action

### Viewing Task Details

Each task card displays:
- Task title and description
- Status icon (circle for To Do, clock for In Progress, alert for Review, checkmark for Completed)
- Priority badge with color coding
- Status badge
- Due date (if set)
- Quick action buttons based on current status

## Uploading Images

### Adding Images to a Job

1. Open the job detail page
2. Navigate to the **Images** tab
3. Add an optional caption for context
4. Click **Select Image** or use the file input
5. Choose an image file (JPG, PNG, GIF, WebP)
6. The file will automatically upload
7. View uploaded images in the gallery below

### Image Requirements

- **Maximum file size**: 10MB
- **Supported formats**: JPG, PNG, GIF, WebP
- **Recommended resolution**: At least 1920×1080 for high quality

### Image Gallery

- Images are displayed in a responsive grid
- Click on any image to view it full size
- Captions appear below each image when provided

### Best Practices for Images

- **Progress photos**: Take regular photos to document construction progress
- **Issue documentation**: Photograph any problems or defects for records
- **Before/after**: Capture before and after shots of major work
- **Safety compliance**: Document safety measures and equipment
- **Quality control**: Record completed work for quality assurance

## Generating Reports

### Accessing Reports

1. Click **Reports** in the sidebar navigation
2. View available jobs for reporting
3. Click **Generate Report** next to any job

### Report Types

The system supports multiple report templates:

1. **Daily Report**
   - Summary of daily activities
   - Tasks completed during the day
   - Progress updates and notes

2. **Task Summary**
   - Overview of all tasks
   - Status breakdown
   - Priority distribution
   - Assignment information

3. **Progress Report**
   - Detailed progress tracking
   - Timeline visualization
   - Milestone completion
   - Overall project status

4. **Image Documentation**
   - Collection of all uploaded images
   - Organized by date
   - Includes captions and metadata

### Exporting Reports

Reports can be exported in multiple formats:
- **PDF**: For printing and sharing
- **Excel**: For data analysis
- **Web view**: For online viewing

*Note: PDF export functionality is planned for a future update.*

## User Roles and Permissions

### User Role

Standard users can:
- View all jobs and tasks
- Create and update tasks
- Upload images
- Generate reports
- Update their own profile

### Admin Role

Administrators have additional privileges:
- Create and edit jobs
- Delete jobs and tasks
- Manage user assignments
- Access system settings
- View all user activity

### Assigning Tasks to Team Members

1. When creating or editing a task
2. Click the **Assign To** dropdown
3. Select a team member from the list
4. The assigned user will see the task in their view

## Tips and Best Practices

### Project Organization

- Create jobs for each distinct construction project or phase
- Use clear, descriptive job titles
- Keep job descriptions concise but informative
- Update job status regularly to reflect current state

### Task Management

- Break down large work items into smaller, manageable tasks
- Set realistic due dates
- Use priority levels to focus team attention
- Update task status as work progresses
- Add detailed descriptions for complex tasks

### Documentation

- Upload images regularly to create a visual record
- Add captions to images for context
- Take photos from multiple angles
- Document both progress and issues
- Keep a consistent naming convention

### Communication

- Use task descriptions to communicate requirements
- Update task status to keep team informed
- Generate regular reports for stakeholders
- Review completed tasks for quality assurance

### Mobile Usage

Mantodeus Manager is fully responsive and works on:
- Desktop computers
- Tablets
- Smartphones

Access the application from the field using your mobile device to:
- Update task status on-site
- Upload photos immediately
- Check task assignments
- View job information

## Keyboard Shortcuts

- **ESC**: Close dialogs and modals

## Troubleshooting

### Images Not Uploading

- Check file size is under 10MB
- Ensure file format is supported
- Verify internet connection
- Try a different browser

### Can't See Recent Changes

- Refresh the page
- Clear browser cache
- Check internet connection

### Task Status Not Updating

- Ensure you have permission to update the task
- Refresh the page
- Check internet connection
